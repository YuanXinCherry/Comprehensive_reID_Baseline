# [ThermalWorld](http://www.zefirus.org/ThermalGAN/)

This is a cross-modality re-id dataset with thermal-color image pairs. Pixel level annotations are provided.

# 简介

这是一个跨模态的重新识别数据集与热彩色图像对。提供像素级注释。
